# Node-Proxy
A reverse proxy build on Node.js

# Dependancies
Node Version >= v8.\*.\*

# Download & Installation
- Using source code
```git clone https://github.com/danielsss/node-proxy.git```

- Install dependancies
```cd ./node-proxy && npm install```
